# 🛍️ Customer Shopping Behavior Analysis

A data analytics project that explores customer shopping patterns using Python, PostgreSQL, and Power BI. The project analyzes 3,900 customer transactions to uncover insights into purchasing behavior, customer segmentation, product performance, and subscription trends.

## 📌 Project Overview

This project focuses on transforming raw transactional data into actionable business insights through data cleaning, SQL analysis, and interactive visualizations. The analysis helps identify customer preferences, spending habits, and opportunities for improving business performance.

## 🛠️ Tech Stack

- **Python** (Pandas, NumPy)
- **PostgreSQL**
- **Power BI**
- **Jupyter Notebook**

## 📂 Dataset

The dataset contains **3,900 customer purchase records** with **18 features**, including:

- Customer demographics (Age, Gender, Location)
- Purchase information (Item Purchased, Category, Purchase Amount)
- Shopping behavior (Discounts, Previous Purchases, Review Ratings)
- Subscription status
- Shipping type
- Purchase frequency
- Seasonal trends

## 🔄 Data Preprocessing

The data was cleaned and prepared using Python by:

- Handling missing values in review ratings
- Standardizing column names
- Creating derived features such as:
  - `age_group`
  - `purchase_frequency_days`
- Removing redundant columns
- Loading the cleaned dataset into PostgreSQL for SQL-based analysis

## 📊 SQL Analysis Performed

Business questions answered include:

- Revenue generated by gender
- High-spending customers who used discounts
- Top-rated products
- Comparison of Standard vs Express shipping purchases
- Subscriber vs Non-subscriber spending analysis
- Products most dependent on discounts
- Customer segmentation (New, Returning, Loyal)
- Top products within each category
- Relationship between repeat purchases and subscriptions
- Revenue contribution by age group

## 📈 Power BI Dashboard

An interactive dashboard was created to visualize:

- Sales distribution
- Customer demographics
- Subscription trends
- Product category performance
- Revenue insights
- Customer segments
- Purchase behavior patterns

## 💡 Key Business Insights

- Subscribers contribute significantly to overall revenue.
- Loyal customers exhibit higher purchase frequency.
- Certain products rely heavily on discounts for sales.
- High-rated products present opportunities for targeted marketing.
- Revenue varies across customer age groups and shipping preferences.

## 🚀 Business Recommendations

- Introduce stronger customer loyalty programs.
- Encourage subscriptions through exclusive benefits.
- Optimize discount strategies to maintain profitability.
- Promote top-performing products in marketing campaigns.
- Target high-value customer segments with personalized offers.
